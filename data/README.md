# DS593 data

## Data Overview

### 1. ds593_dataset_full.txt
This file contains all the raw course materials combined into a single text file.

### 2. ds593_dataset_pages.csv
This file splits the full text into page-level segments.  
Each row represents one page from the lecture materials.

### 3. ds593_chunks.csv
This file further splits the text into smaller chunks.  
These chunks are used for retrieval in the RAG system.
It was divided into a total of 372 chunks.

### 4. ds593_chunks_with_embeddings.pkl
This file stores the embeddings of each chunk.  
It is used for semantic similarity search during retrieval.

### 5. evaluation_questions.csv
This file contains the 50 test questions created based on the course materials.

### 6. evaluation_results.csv
This file stores the answers generated by both the baseline model and the RAG model.

### 7. evaluation_results_fill.csv
This file contains manually labeled results (Yes / Partial / No) for evaluation.  
It is used to compute accuracy and perform error analysis.
